## 🧭 Overview

GFG × RIT is a full-stack community platform built for students passionate about competitive programming, DSA, and software development. It combines an elegant, animated frontend with a functional backend that powers real-time leaderboards, a remote code execution engine, and a gamified learning experience.

The platform is designed to make learning feel less like a chore and more like a game — from interactive roadmaps to a literal DSA racing mini-game.

## ✨ Features

<details>
<summary><b>🏠 Home Page</b></summary>
<br/>

- Cinematic video overlay on first load
- Scroll-driven **Framer Motion** animations — curved scrolling text path and image scattering effect
- Interactive **"Our Mission"** modal
- Club Coordinators gallery with **3D-tilted hover cards**

</details>

<details>
<summary><b>💻 Interactive Code Runner</b></summary>
<br/>

- Full in-browser IDE powered by **CodeMirror 6**
- Syntax highlighting, auto-completion, and bracket matching
- Multi-language support: **JavaScript, Python, C++, Java**
- JS executes securely client-side via `new Function()`; other languages run on the backend
- Automatic **test case validation** with pass/fail visual badges

</details>

<details>
<summary><b>🏎 DSA Racer Game</b></summary>
<br/>

- Browser-based driving mini-game built for education
- Dodge incorrect answers to DSA questions drifting down as obstacles
- Arrow key + tap controls, collision detection, score counter
- Increasing difficulty speed mechanics
- Crash sound effects via the **Web Audio API**

</details>

<details>
<summary><b>🗺 Learning Roadmaps</b></summary>
<br/>

- Interactive tabbed learning paths: **DSA, Web Dev, AI/ML, Data Science, Cyber Security**, and more
- Vertical animated timelines with expandable milestone cards

</details>

<details>
<summary><b>📚 Courses Center</b></summary>
<br/>

- Curated course directory categorized by domain
- Detailed cards with curriculum, difficulty, and duration
- Direct links to **GFG, LeetCode, HackerRank, freeCodeCamp, Coursera**

</details>

<details>
<summary><b>🏆 Leaderboard</b></summary>
<br/>

- Top 20 users ranked by accumulated **XP**
- Dynamic trend indicators and growth metrics
- **DiceBear SVG avatars** auto-generated per user

</details>

<details>
<summary><b>🔐 Auth & Protected Routes</b></summary>
<br/>

- `AuthContext`-based authentication flow
- Community, Leaderboard, and Settings pages are fully protected
- Unauthenticated users are automatically redirected to `/auth`

</details>

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

| Tool | Version | Download |
|------|---------|----------|
| Node.js | v18+ | [nodejs.org](https://nodejs.org/) |
| Git | Latest | [git-scm.com](https://git-scm.com/) |

---

### Installation

**1. Clone the repository**

```bash
git clone <your-repository-url>
cd gfg-web
```

**2. Set up and run the Frontend**

```bash
cd _vite_scaffold
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

**3. Set up and run the Backend** *(required for Python / C++ / Java execution)*

```bash
cd backend
npm install
npm run dev
```

API available at [http://localhost:3000](http://localhost:3000).

## 📡 API Reference

Base URL: `http://localhost:3000`

| Method | Endpoint | Description |
|--------|----------|-------------|
| `GET` | `/api/leaderboard` | Fetch top 20 users ranked by XP |
| `GET` | `/api/daily-question` | Get the rotating DSA question of the day |
| `GET` | `/api/users/:username` | Fetch a specific user's profile |
| `POST` | `/api/users/:username/xp` | Award XP points to a user |
| `POST` | `/api/users/:username/streak` | Update a user's activity streak |
| `POST` | `/api/execute` | Execute code remotely (Python, C++, Java) |

<details>
<summary><b>POST /api/execute — Request Body</b></summary>
<br/>

```json
{
  "language": "python",
  "version": "3.x",
  "files": [
    {
      "name": "main.py",
      "content": "print('Hello, World!')"
    }
  ]
}
```

**Response**

```json
{
  "stdout": "Hello, World!\n",
  "stderr": ""
}
```

</details>
